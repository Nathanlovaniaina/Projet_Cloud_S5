# Back End

Ce dossier contient l'API REST développée avec Java Spring Boot pour le projet Signalement Travaux Routiers.

## Modules :
- **Authentification** : Setup Spring Boot, configuration Firebase, connexion PostgreSQL, APIs pour inscription/connexion, gestion des sessions, modification profil, limite tentatives, déblocage utilisateurs, documentation Swagger
- **Signalements** : APIs pour créer, récupérer, modifier signalements
- **Synchronisation** : APIs pour synchro avec Firebase (récupération et envoi)
- **Statistiques** : API pour récapitulatif
