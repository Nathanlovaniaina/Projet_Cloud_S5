# Base de Données

Ce dossier contient la configuration et les scripts pour la base de données PostgreSQL du projet Signalement Travaux Routiers.

## Tâches associées :
- Configuration du conteneur Docker PostgreSQL
- Création du MCD (Modèle Conceptuel de Données)
- Création des tables pour les utilisateurs
- Création des tables pour les signalements
- Création des tables pour le suivi des tentatives de connexion
