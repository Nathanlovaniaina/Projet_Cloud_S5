# Front End Mobile

Application mobile Ionic avec Vue.js pour signaler les travaux routiers.

## Fonctionnalités :
- Setup : initialisation Ionic, configuration Firebase SDK, Leaflet
- Authentification : écran de connexion Firebase
- Carte : intégration Leaflet avec OpenStreetMap, affichage signalements
- Signalement : géolocalisation, interface création sur carte
- Filtre : "Mes signalements uniquement"
- Build : configuration et génération APK Android
