# Front End Web

Application React pour la gestion des signalements de travaux routiers via interface web.

## Fonctionnalités :
- Authentification : composants de connexion, inscription, gestion sessions, modification profil
- Carte : intégration Leaflet pour afficher Antananarivo, marqueurs signalements, survol
- Visiteur : page avec carte et récapitulatif
- Manager : gestion signalements, synchro Firebase, déblocage utilisateurs, édition statuts
- Design : responsive, navigation et routing
